Setup Guide

1. Add Photos: Uncomment the <img> line inside each card-image div and add your photo filename. Example: <img src="john.jpg" alt="John">

2. Add Audio Files: Put all 15 MP3 files in the same folder as this HTML. Update each src="song1.mp3" to match your actual filenames (like miss_summer.mp3).

3. Customize Content: Replace the placeholder text with real info - names, descriptions, fun facts, and song titles.

4. Upload Online: When ready to go live, upload this HTML file + all MP3s + all photos to your hosting (GitHub Pages, Netlify, etc.). Keep everything in the same folder!
